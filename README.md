LuxiCalc: A Modern Android Calculator
LuxiCalc is a sleek and intuitive calculator application built for Android using the latest mobile development technologies. Designed with a clean, user-friendly interface, it provides a seamless experience for performing basic arithmetic operations. The app is developed entirely with Jetpack Compose, showcasing a modern, declarative approach to UI development in Kotlin.
Key Features:
•
Modern UI: A beautiful and responsive user interface built with Jetpack Compose.
•
Core Arithmetic Operations: Supports addition, subtraction, multiplication, and division.
•
Expression History: Keep track of your previous calculations with a dedicated history screen.
•
User-Friendly Design: A simple and intuitive layout that makes calculations quick and easy.
Technology Stack:
•
Kotlin: The official language for Android development.
•
Jetpack Compose: A modern toolkit for building native Android UI.
•
exp4j: A powerful open-source library for evaluating mathematical expressions.
•
Material Design 3: Adheres to the latest design guidelines from Google for a consistent and visually appealing user experience.
